  • flex

    • flex: auto: flex: 1 1 auto, 参考块的宽高, 尽量的占据剩余空间, 如果空间小于自身尺寸, 则缩小以适应空间
    • flex: initial: flex: 0 1 auto, 默认值, 不会尽量的占据剩余空间
    • flex: none: flex: 0 0 auto, 完全不会改变尺寸, 只参考自身宽高
    • flex: 正数: flex: 正数 1 0
    • 可以设置min-width, min-height, 让flex元素不会缩小到这个值下面
